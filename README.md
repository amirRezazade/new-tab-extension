# 🚀 Nova New Tab

> یه صفحه تب جدید سفارشی برای کروم با قابلیت مدیریت لینک‌ها، تب‌ها، گروه‌ها و شخصی‌سازی کامل

![Nova New Tab Preview](./screenshots/preview.png)

---

## ✨ قابلیت‌ها

### 📌 مدیریت لینک‌ها

- ساخت **تب‌های** مختلف (مثلاً Work، Personal، Reading)
- ساخت **گروه‌های لینک** داخل هر تب
- اضافه، ویرایش و حذف لینک‌ها
- نمایش **favicon** سایت‌ها به‌صورت خودکار
- **درگ و دراپ** برای جابجایی لینک‌ها، گروه‌ها و تب‌ها

### 🗂️ چیدمان ستونی

- صفحه به **۴ ستون** تقسیم می‌شه
- می‌تونی گروه‌ها رو به هر ستونی درگ کنی
- می‌تونی چندتا گروه رو زیر هم تو یه ستون بذاری

### 🔍 جستجو

- **جستجوی گوگل** مستقیم از سرچ‌بار بالای صفحه
- **جستجو بین لینک‌ها** با دکمه FAB (گوشه پایین راست)

### 🎨 شخصی‌سازی

- تغییر **والپیپر** از تصاویر آماده یا آپلود عکس دلخواه
- انتخاب **رنگ accent** و **رنگ متن**
- تنظیم **میزان بلور** و **شفافیت** کارت‌ها

### ⏰ ابزارها

- نمایش **ساعت و تاریخ **
- نمایش **آب‌وهوا** بر اساس شهر انتخابی

### ⚙️ تنظیمات

- باز شدن لینک‌ها در **تب جدید یا همون تب**
- نمایش/مخفی کردن **سرچ‌بار**
- **Export** داده به فایل JSON
- **Import** داده از فایل JSON

### ☁️ ذخیره‌سازی

- همه داده‌ها با **chrome.storage.sync** ذخیره می‌شن
- تب‌ها و لینک‌ها بین دستگاه‌های مختلف **sync** می‌شن

## 🚀 نصب

### روش اول: نصب به‌عنوان Developer (برای توسعه‌دهنده)

1. این مخزن رو clone کن یا دانلود کن
2. مرورگر کروم رو باز کن
3. برو به `chrome://extensions`
4. گوشه بالا راست، **Developer mode** رو روشن کن
5. روی **Load unpacked** کلیک کن
6. پوشه پروژه رو انتخاب کن
7. یه تب جدید باز کن — Nova New Tab آماده‌ست!

### روش دوم: نصب از Chrome Web Store

## 📝 نکات مهم

- والپیپرهای preset بین دستگاه‌ها sync می‌شن، ولی والپیپر آپلودی فقط local ذخیره می‌شه
- اگه از VPN استفاده می‌کنی، شهرت رو دستی تو Settings وارد کن
- برای بهترین نتیجه در جستجوی شهر، اسم شهر رو فارسی بنویس
